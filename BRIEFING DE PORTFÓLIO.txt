Instruções para o Programador — Como Montar o Portfólio
Estrutura Sugerida da Página
A página de portfólio deve ter uma única página scrollável (one-page) com as seguintes seções:

•	Hero / Apresentação: nome, título profissional, breve bio, link para o Behance
•	Sobre o diferencial de IA: bloco destacado explicando o uso de IA generativa
•	Projetos: cards ou seções para cada projeto (6 no total)
•	Contato: formulário ou links de contato

Como Apresentar os Slides
Cada projeto deve ter:
•	Uma galeria de slides em carrossel ou grid (usar os slides indicados neste briefing)
•	O nome do cliente, tipo de projeto e tags (Evento, Campanha, IA, etc.)
•	Uma descrição curta do projeto (2-3 linhas)
•	Destaques em bullets

Badge de Inteligência Artificial
  ★ REGRA IMPORTANTE: Todo slide que contiver uma imagem criada com IA deve ter um badge/tag visível 'IA Generativa' ou 'Criado com IA'. Isso é um diferencial competitivo e deve estar em destaque.

Slides que obrigatoriamente precisam do badge de IA:
•	FTD Travessia — Slide 4: ponte cenográfica
•	FTD Travessia — Slide 6: mock-up de celular com QR Code
•	Zurich/Carrefour — Slide 5: ecobag com pinceladas (adotada pelo cliente!)
•	Zurich/Carrefour — Slide 9: painel Lab 360° de Proteção
•	Zurich/Carrefour — Slide 14: lata Zurich Z-Energy
•	Amigoz — Slide 8: mock-up de smartphone
•	Amigoz — Slides 15-16: super-heróis para os perfis da campanha

Como Exportar os Slides
Os slides devem ser exportados como imagens PNG ou JPG diretamente do PowerPoint:
•	No PowerPoint: Arquivo > Exportar > Alterar Tipo de Arquivo > PNG ou JPEG
•	Resolução recomendada: 1920x1080px (Full HD) ou maior
•	Exportar apenas os slides indicados neste documento (número exato listado em cada projeto)
•	Nomear os arquivos de forma organizada: ex. 'ftd-travessia-slide04-ponte-ia.png'
